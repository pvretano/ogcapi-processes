[[req_deploy-replace-undeploy_deploy_unsupported-media-type]]
[requirement]
====
[%metadata]
identifier:: /req/deploy-replace-undeploy/deploy-unsupported-media-type

[.component,class=part]
--
If the server does not support the Content-Type header associated with the request body, the code of the response SHALL be `415 Unsupported Media Type`.
--

[.component,class=part]
--
The content of that response SHALL validate against the following OpenAPI schema fragment:

[source,yaml]
----
include::../../../../../../openapi/schemas/common-core/exception.yaml[]
----
--

[.component,class=part]
--
The `type` of the exception SHALL be “https://www.opengis.net/def/exceptions/ogcapi-processes-2/1.0/unsupported-media-type”.
--
