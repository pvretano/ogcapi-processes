[[req_job-management_definition_get-op]]
[requirement]
====
[%metadata]
identifier:: /req/job-management/definition-get-op

[.component,class=part]
--
For every jobs (using method: GET on path: `/jobs/{jobID}`),
the server SHALL support the HTTP GET operation at the path `/jobs/{jobID}/definition`.
--

[.component,class=part]
--
The parameter `jobID` resolves to a valid job as per <<OAProc-1,OGC API - Processes - Part 1: Core, Conformance Class 'Core'>>.
--
