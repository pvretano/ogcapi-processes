[[req_openeo_create_body]]
[requirement]
====
[%metadata]
identifier:: /req/openeo/create-body

[.component,class=part]
--
The media type `application/json` SHALL be used to indicate that request body contains a processes description
encoded as an <<rc_openeo,OpenEO Process Graph>>.
--

[.component,class=part]
--
The body SHALL conform to the
https://raw.githubusercontent.com/Open-EO/openeo-processes/master/meta/subtype-schemas.json#/definitions/process-graph[OpenEO Process Graph schema].
--

====
